#### Possible convertions:
- g/L to mol/L
- mol/L to g/L 
- g to mmol
- mmol to g
- g/mol to M in solution

### How to run
Install requirements.txt
```
pip install -r requirements.txt
```
Run
```
python chem_calculator.py
```
